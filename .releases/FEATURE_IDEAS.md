# Идеи для будущих функций (Feature Roadmap)

Этот список содержит планируемые и потенциальные фичи для развития **Findo Finance Bot**.

---

## 1. Интеллектуальное управление подписками (AI + Payments)
Сейчас база «Ожидаемых платежей» доступна только через команды. Нужно дать AI-агенту инструменты для управления этими данными.
- **Инструменты:** `add_expected_payment`, `delete_expected_payment`, `list_payments`.
- **Кейс:** «Запиши подписку на Netflix за 15 евро 20-го числа каждого месяца».

## 2. Визуализация и графики (Charts via Telegram)
Добавить визуальную составляющую к сухим цифрам отчетов.
- **Технология:** Использование `matplotlib` или `plotly` для генерации PNG-графиков.
- **Кейс:** Круговые диаграммы расходов по категориям, бар-чарты динамики расходов по неделям/месяцам.

## 3. Финансовые цели и «Копилки»
Система накопления на конкретные цели с отслеживанием прогресса.
- **Функционал:** Создание целей с целевой суммой и дедлайном.
- **Кейс:** «Отложи 50 евро в копилку на отпуск». Бот показывает: «Собрано 450/1000€ (45%)».

## 4. Продвинутая аналитика и сравнение
Использование LLM для поиска аномалий и советов по оптимизации.
- **Кейс:** «Почему я потратил в этом месяце больше, чем в прошлом?». Бот анализирует: «В этом месяце было 5 походов в рестораны против 2 в прошлом».

## 5. Мультивалютность с авто-конвертацией
Автоматический перевод трат в разных валютах в «базовую» валюту пользователя.
- **Интеграция:** API курсов валют (например, ExchangeRate-API).
- **Кейс:** Запись траты в лирах или долларах с автоматическим пересчетом в EUR/RUB по курсу дня.

## 6. Групповые (семейные) бюджеты
Общий доступ к одной таблице для нескольких пользователей.
- **Функционал:** Идентификация автора транзакции в Google Sheets и раздельная статистика.
- **Кейс:** Семейный бюджет, где видно, кто и сколько внес/потратил.

## 7. «Умные» лимиты и проактивные алерты
Бот должен предупреждать пользователя заранее, а не только по запросу.
- **Кейс:** Уведомление, когда бюджет категории или зоны подходит к концу (осталось 10% или 5%).

## 8. Парсинг банковских уведомлений
Упрощение ввода данных за счет копирования текста из банковских приложений.
- **Кейс:** Пользователь пересылает СМС/Пуш от банка, бот автоматически вычленяет сумму, магазин и категорию.
